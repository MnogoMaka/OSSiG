# OSSIG
## Основные возможности
- **Проверка соответствия** (`/task/accordance`):  
  Анализ соответствия документов реальному состоянию автомобиля с использованием YOLO-модели классификации
- **Оценка загрязнения** (`/task/pollution`):  
  Определение уровня загрязнения транспортного средства (чистое/грязное)
- **Анализ груза** (`/task/cargo`):  
  Обнаружение ТС и классификация груза 
- **Обнаружение мошенничества** (`/task/fraud`):  
  Сравнение изображений входа/выхода с использованием внешнего API для выявления несоответствий
# Заполните API_URL в .env файле
